What is an Operating System ?
This is consisting of the central software managing a computer’s resources and, all the accompanying standard
software tools, such as command-line interpreters, graphical user interfaces, file utilities, and editors.

Kernel
This the central software that manages and allocates computer resources (CPU, and RAM and Devices).
