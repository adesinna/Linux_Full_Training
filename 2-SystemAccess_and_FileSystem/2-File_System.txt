File system is use by the os to manage files. The system controls how data is
saved or retrieved

/                  # this is the root directory

/boot              # Contains file that is used by the boot loader (grub.cf)

/root              # root user home directory

/dev               # system devices like disk, cdrom, keyboard

/etc               # configuration files

/usr/bin          # everyday user commands

/usr/sbin         # System/filesystem commands

/opt              # Optional add-on applications (Not part of the OS apps)

/proc             # running processes (only exist in the memory)

/usr/lib          # C library needed by commands and apps

/tmp              # Directory for temporary files

/var              # System logs

/run              # stores run time files like PID

/mnt              # To mount external files

/media            # for cdrom mounts
