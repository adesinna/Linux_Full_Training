In linux, many user connects to the linux system, we have root users, regular users, service account and system accounts.

id user

who
The who command shows you exactly who is logged into the computer at this very second.

last
The last command looks backward in time. It shows a list of all successful logins and logouts that have happened recently.


/etc/passwd
This contains basic information about the users

/etc/shadow
This contains users passwords, these are encrypted

/etc/groups
This contains information about the user groups
